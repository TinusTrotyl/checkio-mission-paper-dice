This mission is an advanced 'Identify Block'.

You have to write a code to check a development drawing of the dice.

The input data represents a development drawing of a die by a list of strings. These strings are the same in lenght and consist of 1 to 6 digits or spaces only. The digits represent the faces of the dice.

  • If the development is right, return True.
  • For the followng cases, return False:
    • If you can not construct a cube.
    • When the sum of two oposite sides does not equal 7.

Example:

paper_dice(['  1  ',
            ' 235 ',
            '  6  ',
            '  4  ']) == True

paper_dice(['    ',
            '12  ',
            ' 36 ',
            '  54',
            '    ']) == True

paper_dice(['123456']) == False
paper_dice(['123  ',
            '  456']) == False

Input: A list of String.

Output: The answer as a boolean.

Precondition:

  • all(len(row) == len(paper[0]) for row in paper)
  • all((cell in '123456 ' for cell in row) for row in paper)
